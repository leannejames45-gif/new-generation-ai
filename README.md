# AI Video Generator

A modern, responsive web application for generating videos from text descriptions using AI technology.

## Features

- **Text-to-Video Generation**: Transform text descriptions into video content
- **Multiple Styles**: Choose from realistic, animated, abstract, and fantasy styles
- **Real-time Preview**: See your generation come to life with live previews
- **History Panel**: Review your previously generated videos
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Technology Stack

- **React** with TypeScript for the frontend
- **Tailwind CSS** for styling
- **Shadcn/ui** components for consistent UI
- **Vite** for fast development and building
- **AI Integration** for video generation (placeholder for actual AI service)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:5173](http://localhost:5173) to view the app

## Project Structure

```
src/
```
